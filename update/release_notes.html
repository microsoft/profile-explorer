Version 0.3.0
+ Show recent files list in the File menu (up to 10 files remembered)
+ Reload current file in the File menu, or press F5
+ Auto-reload of the current file, enabled by default
    It monitors the current IR file and shows a message box asking if the file should be reloaded
+ The current IR file is no longer kept open, was preventing updating the file
    If it’s below 50 MB it’s read completely in memory
+ Highlighting for uses/definition of a variable works now with non-SSA tuples
    Since it’s based on the symbol (var name), it will highlight all defs/uses
    The reference panel also shows all refs for such cases instead of 0 SSA uses
+ Improved parsing of post-lower tuples and some other verbose info
+ Syntax highlighting for many x86/x64 opcodes
+ A new panel that shows the pass (debug) output for a section
    View the text either before or after a pass
    Variables (operands) are identified and preview/go-to-definition can be done. Very useful to see where a logged action was applied
+ Notes panel also identifies variables
+ Notes panel has a new, default mode: per-document (session) notes
+ Source file panel is working in both directions (click on tuple, highlight source line, click on line, highlight all tuples)
+ View -> Side-by-side panels creates 2 Flow Graph and Reference panels bound to the left/right documents
    This needs two sections opened side-by-side, with or without diff mode enabled
+ Remember window size, position, monitor, theme
+ Open file in new window in the File menu
* Cancel long running Graphviz layout
* Flow graph changes when switching between multiple opened sections
* Flow graph remembers when switching between multiple opened sections
* Fixed several UI bugs and UTC parsing issues
* New icon that's more visible on the taskbar
