The Profiling window is the main way for loading and recording profiling traces. To load a profiling trace file (ETL file produced by tools like xperf and WPR), there are several ways:

  • Select Profling menu -> Load Profile and specify a file in the Profile data file field.

  • Select File menu -> Open File (Ctrl+O). After picking a file, this will show the Profiling window with the trace file path filled in.

In the Profiling window, once a trace file is selected in the Profile data file field, a summary of all processes found in the trace is displayed, with the list sorted by weight (number of samples per app).

Select the process to load from the list and click the Load Profile button (or simply double-click).
