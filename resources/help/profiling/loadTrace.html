To load a profiling trace file (ETL file produced by tools like xperf and WPR), there are two main ways:

  • Through the Profiling window, displayed selecting Profiling menu -> Load Profile

  • As a regular file opened using File menu -> Open File (Ctrl + O). After selecting a file, this will show the Profiling window mentioned above with the trace file path filled in.
