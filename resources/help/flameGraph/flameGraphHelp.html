Keyboard shortcuts
Mouse shortcuts

Click

Selects a node, updates the Details panel and selects the function in the other profiling panels if Sync is enabled

Double-Click

Enlarges a node and all child nodes to fill the view
Back Undoes the previous node enlarging or set as root operation
Alt + Double-Click Sets the selected node as the root node of the flame graph
Click + Drag Done on an empty part of the panel moves the flame graph
Scroll Moves the flame graph in the vertical direction
Shift + Scroll Moves the flame graph in the horizontal direction
